# **Memory Cards Game**

A simple memory-matching game built with [HTML/CSS/JS]. While the core logic follows a tutorial, all SVGs and visual assets were designed by me. This project was a way to practice frontend implementation and asset creation.

- **Features:** Card matching, score tracking, responsive design
- **Skills demonstrated:** SVG design, DOM manipulation, event handling
- **Note:** Tutorial-based, but with custom artwork
